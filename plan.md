# TalentFlow AI

## Goal
Build an AI-powered Applicant Tracking System (ATS) that helps recruiters manage jobs, screen resumes, and track candidates efficiently.

## Users
- Candidate
- Recruiter
- Admin

## Core Features
- User Authentication
- Job Posting
- Resume Upload
- AI Resume Analysis
- Application Tracking
- Recruiter Dashboard
- Candidate Dashboard
- Analytics
- Responsive Design

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- Auth.js
- Vercel
