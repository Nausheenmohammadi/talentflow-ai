# TalentFlow AI

An AI-powered Applicant Tracking System (ATS) built with Next.js to help recruiters manage hiring and candidates apply for jobs efficiently.

---

## 🚀 Features

- AI Resume Screening
- Smart Candidate Ranking
- Recruiter Dashboard
- Candidate Dashboard
- Job Management
- Responsive Design

---

## 🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Git & GitHub
- Vercel
